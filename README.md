# Batalla Naval - Proyecto Final 🚢

Este proyecto es un simulador del clásico juego de estrategia naval, desarrollado como trabajo final para la cursada de ingeniería de software. El objetivo principal fue demostrar la aplicación de arquitecturas robustas y lógica de programación avanzada en un entorno web.

## 🛠️ Stack Tecnológico
* **Backend:** PHP para la lógica del servidor, gestión de sesiones y procesamiento de reglas.
* **Frontend:** JavaScript (Vanilla) y CSS para una interfaz de usuario dinámica y responsiva.
* **Base de Datos:** MySQL para la persistencia de datos, gestión de usuarios y sistema de ranking.

## ✨ Características Principales
* **Arquitectura Orientada a Objetos:** Implementación de clases especializadas para diferentes tipos de embarcaciones y tableros.
* **Principios SOLID:** Aplicación de buenas prácticas de diseño para asegurar un código mantenible, escalable y fácil de testear.
* **Patrones de Diseño:** Uso estratégico de patrones como **Strategy** y **State** para gestionar el comportamiento de la flota y el flujo de los turnos.
* **Sistema de Ranking:** Base de datos integrada para almacenar puntajes y visualizar a los mejores jugadores competitivos.
* **Interacción Dinámica:** Manejo de eventos mediante JavaScript para procesar ataques y actualizaciones visuales sin necesidad de recargar la página.

## 🚀 Instalación y Configuración
1. **Servidor Local:** Requiere un entorno como **XAMPP** o Apache para la ejecución de los scripts PHP.
2. **Base de Datos:** Importar el esquema SQL proporcionado en el directorio correspondiente utilizando herramientas como PHPMyAdmin.
3. **Conexión:** Configurar los parámetros de acceso a la base de datos en el archivo de conexión del backend.
4. **Ejecución:** Alojar el proyecto en el directorio `htdocs` y acceder a través de `localhost/batalla-naval`.
